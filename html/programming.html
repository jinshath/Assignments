PROGRAMMING LANGUAGES

A Programming language is a formal language designed to communicate instructions to machine, particularly a computer. Programming languages can be used to create programs that control the behaviour of a machine and to express algorithms precisely.

C++

C++ is a general-purpose programming language. It has imperative, object-oriented and generic programming features, while also providing facilities for low-level memory manipulation.

C

C is a general-purpose, imperative computer programming language, supporting structured programming, lexical variable scope and recursion, while a static type system prevents many unintended operations

JAVA

Java is a general-purpose computer programming language that is concurrent, class-based, object-oriented,[14] and specifically designed to have as few implementation dependencies as possible

Oxygen and life

Oxygen is a chemical element with symbol O and atomic number 8. It is a member of the chalcogen group on the periodic table and is a highly reactive nonmetal and oxidizing agent that readily forms oxides with most elements as well as other compounds.[4] By mass, oxygen is the third-most abundant element in the universe, after hydrogen and helium.
